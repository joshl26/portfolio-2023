## Prerequisites

- npm >=8.0.0
- node >=17.0.0

_YOU WILL NEED TO SUPPLY YOUR OWN TEXTURE MAPS AND MODELS_

## Install

```sh
npm install
```

## Usage

```sh
npm run start
```

## Run tests

```sh
npm run test
```
